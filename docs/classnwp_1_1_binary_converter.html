Detailed Description

The BinaryConverter class.

変数とバイナリの変換を行いたいメソッド群 変換可能なのは

  • triviallycopyableな型の値
  • triviallycopyableな型の配列
  • triviallycopyableな型の動的コンテナ(std::begin/std::endの呼び出しが可能であること)
  • nwp::utility::Dummy<>
    Note
    C++20でstd::bit_castが来たら実装しなおす(constexpr化できる)
