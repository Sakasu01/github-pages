Detailed Description

template<std::size_t n>
class nwp::BinaryData< n >

固定長のバイナリデータ

データフォーマットを異なるプログラムで共有するときに便利かな...とか思ったり

Template Parameters
nデータ長[byte]
例
using Fmt = std::tuple<uint8_t, uint16_t, float>;
BinaryData<8> data;
// serialize
data.toBinary<Fmt>(12, 34, 3.14f); // u8, u16, floatにエンコードされる
// deserialize
auto [x, y, z] = data.fromBinary<Fmt>(); // u8, u16, floatとしてデコードする
