Detailed Description

template<class Method>
class peripheral::UART< Method >

UARTを用いてシリアル通信をするクラス

送受信をDMAで行います 内部で保持されるバッファサイズを超えたデータの扱いは以下の通りです

操作 動作 対処法
送信 余剰分のデータは送信されず破棄 ボーレートを超えたデータの送信を試みない
受信 先頭からバッファサイズ分のデータが消滅 不具合を引き起こす前にデータを取り出す
Template Parameters
MethodUARTMethodを継承したクラス マイコンごとの実装がなされていること
使用例
peripheral::UART<F4::UARTMethod> serial(F4::uart1); // F4::UARTMethodの部分は適宜変更すること
serial.config(F4::PA2, F4::PA3);
serial.print("Hello World\n");
