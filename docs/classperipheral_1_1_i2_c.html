Detailed Description

template<class Method>
class peripheral::I2C< Method >

I2Cのマスタとして動作するクラス

Note
start/restart/end/write/readを別々の実装にしてくれという主張があると思いますが、
そうでない実装になったのはすべてF3のペリフェラル構造がクソであるためなので、怒りの矛先はSTに向けてください
