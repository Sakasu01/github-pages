Memory Allocator Clue Function.
Adjusts end of heap to provide more memory to
memory allocator, With Simple sanity checks.

This implementation is not thread safe (despite taking a
_reent structure as a parameter).
Since _s_r is not used in the current implementation,
the following messages must be suppressed.

Parameters
struct_reent *_s_r: not in use currently.
ptrdiff_tnbytes: specify bytes to use memory.
Return values
None.
