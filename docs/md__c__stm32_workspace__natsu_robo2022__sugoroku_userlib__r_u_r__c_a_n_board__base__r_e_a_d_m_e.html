RUR_CANBoard_Base

CAN基板周りのベースライブラリです

RUR_CANBoard_MasterとRUR_CANBoard_Slaveの両方で使うものや通信において統一するべきものが入ってます

Description

  • board 基板ごとのオブジェクトコード及び関数コードの定義．またイベントのトリガ設定の実装
  • interface 関数コードの定義やベースクラスの実装．RUR_CANBoard_MasterやRUR_CANBoard_Slaveで派生させる
  • util 通信や全基盤共通なものが入ってる
  • CANClient.hpp 事実上こいつがどの基盤でもCANを操作するクラス

VS.

C++にしたので動作が分かりやすくなってる...かもしれない

ここのインターフェースは全基盤共通にできるので整備性が良くなった

Requirement

  • CANProtocol (C++のやつ)
  • RUR_STM_PeriphLibなど，ペリフェラルをいじるもの

Usage

サブモジュールとしてuserlibに追加してください

Example

MasterかSlaveを参照してください
