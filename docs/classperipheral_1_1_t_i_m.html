TIMクラス

Template Parameters
TIMMethodを継承したクラスマイコンごとの実装がなされていること
使用例
// tim1を100Hzで使用
peripheral::TIM<F4::TIMMethod> tim(F4::TimerClockGenerator<F4::tim1, 100>{}); // F4::TIMMethodの部分は適宜変更すること
// 初期化
tim.config();
// 優先度0(最大)で割り込み設定
tim.configIT(0, []{ ~~ });
//タイマ動作開始
tim.setStart(true);
