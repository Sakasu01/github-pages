Detailed Description

template<class Derived>
class peripheral::DMAMethodInterface< Derived >

DMAの操作インターフェースクラス

DMA<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

Attention
Derived内でpublic指定でChannelInfoを定義すること ChannelInfoはDerivedの単一のコンストラクタ引数であること
実装例
namespace F4 {
class DMAMethod : peripheral::DMAMethodInterface<DMAMethod> {
public:
using ChannelInfo = std::pair<PeriphInfoType<DMA_Stream_TypeDef>, uint32_t>; // 必須
protected:
DMAMethod(ChannelInfo chInfo);
// 各メソッドの実装
};
}
