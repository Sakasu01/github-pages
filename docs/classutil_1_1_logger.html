Detailed Description

template<class Method, class Config, bool Enable>
class util::Logger< Method, Config, Enable >

デバッグ用のロガークラス

出力フォーマットには、FormatOutputterの

  • '$'による安全なフォーマット を実装しています

フォーマットの設定やオブジェクトの生成は別ファイルで行います LoggerConfig_template.hppのコピーをLoggerConfig.hppに改名してinc/等に置いてください

Template Parameters
Method出力先 util::FormatOutputterをCRTP継承していること
Enable出力の有効化
使用例
logging::info.LOG("hogeの値は$です", hoge);
