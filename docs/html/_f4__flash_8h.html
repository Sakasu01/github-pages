◆ FLASH_Sector_NUM

#define FLASH_Sector_NUM   11

使用するセクタの番号を指定します。0~11の数字で指定して下さい。

セクタ0～3には16KB、セクタ4には64KB、セクタ5～11には128KBのデータを格納できます。 最初の方のセクタにはプログラムが書かれている可能性が高いので使わない方が安全です。

◆ FLASH_WriteData()

uint8_t FLASH_WriteData ( const uint16_t  StartDataNum,
void *  pData,
const uint16_t  DataLength 
)

データの書き込みを行います。

書き込みは32bit単位で行われます。uint32_tやfloatの変数や配列、uint8_tの要素数4*Nの配列などが使えます。 事前に1度FLASH_Eraseを実行する必要があり、1度書き込んだ部分には次にセクタを消去するまで2度と書き込む事はできません。

Parameters
StartDataNum32bit単位でセクタ内の何番目からデータを書き込むかを指定(最小は0で、128Kbytesのセクタでは最大0x8000からDataLengthを引いた値までが指定可能)
pData書き込むデータの先頭ポインタを指定
DataLength書き込むデータのサイズを32bit単位で指定
Returns
成功ならSUCCESS(1)、失敗ならERROR(0)を返す
