Detailed Description

template<class T, std::size_t capacity_, bool enableVolatile = false>
class rur::container::StaticQueue< T, capacity_, enableVolatile >

スタック領域のみで動作するQueue(FIFOコンテナ)

決められたバッファサイズ内で動作するQueueです

Attention
std::queueと互換を持たせるために、あえて部内のコーディング規約を無視しています
計算量
操作 計算量
先頭要素へのアクセス O(1)
全要素の両方向への操作 不可
末尾への要素の追加 O(1)
先頭の要素の削除 O(1)
コンテナのコピー/ムーブ O(N)
動的アロケートによるコストを気にしない場合はstd::queueを用いたほうが良いかと(可変長なので)
Note
std::vector::insert/eraseに相当するものがないので、そのうち作ったほうがいいと思う
Template Parameters
Tオブジェクトの型 非void && const修飾されてない destructible(非POD型の場合) を満たす必要があります
maxSizeオブジェクトの最大数 この数を超えて要素を確保することはありません(出来ません)
enableVolatiletrueにすると最適化抑止が少し利いて、割り込みなんかで使えるように
