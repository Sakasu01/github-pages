RUR_STM_PeriphLib

STM32マイコンのペリフェラルライブラリです

後述するRUR_STM_F[3|4]Libとの連携を前提に構築しています

Description

STM32マイコンで用いるペリフェラル(GPIO, ADC, TIM等)及び周辺ライブラリ(ENC, PWM等)をちょめちょめするライブラリです

STM_PeriphLibでは以下を提供します

  • 各種ペリフェラルに要求されるMethodのインターフェース
  • 上記メソッドを使用したペリフェラルクラス
  • 上記メソッドを使用した周辺機器クラス(LED, PWM, Encoder等)

Methodの実装はSTM_F[3|4]Libで提供されます

VS

  • stm32plus
    • F3シリーズの対応がないので、この部では使いづらい？
    • また、がちがちのc++ templateで組まれているので、だいぶ慣れてないと難しいかも？(おまいう)

Requirement

  • STM32SPL
  • RUR_STM_CommonLib

だいぶ調査不足ですが、このライブラリはインターフェース群なので、STMの標準ライブラリをSPLからHALに置き換えてもある程度動くとは思います

CubeMxはたぶん対応できないが、otherモジュールのみうまく変更すれば...

Usage

userlibにgit submoduleでimportして下さい

RUR_STM_F[3|4]Libも別で導入する必要があります(必須ではないが、たぶん入れないと意味がないと思われる)

詳しくはチュートリアルを参照してください

Example

細かくは各peripheralのインターフェースのドキュメントを参照してください

例としてF4のGPIOの場合

peripheral::GPIO<F4::GPIOMethod> led(F4::PA0);
led.config(peripheral::gpio::Mode::out);
led.write(true);
led.write(false);
