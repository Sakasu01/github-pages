Detailed Description

template<class T>
struct util::Ref< T >

参照ラッパークラス

関数に参照渡しする際に明示できます それだけ
下の使用例を見てもらうのが一番いいと思う
std::reference_wrapperとの最大の違いは、コンストラクタにexplicitがついていること

使用例
void f(Ref<int> r) {
r = 3;
}
int x = 0;
// f(x); // コンパイルエラー
f(Ref(x)); // 明示すればおｋ
assert(x == 3);
