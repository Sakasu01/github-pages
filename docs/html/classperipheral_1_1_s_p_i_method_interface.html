Detailed Description

template<class Derived>
class peripheral::SPIMethodInterface< Derived >

SPIの操作インターフェースクラス

SPI<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

実装例
namespace F4 {
class SPIMethod : peripheral::SPIMethodInterface<SPIMethod> {
protected:
// 各メソッドの実装
};
}
