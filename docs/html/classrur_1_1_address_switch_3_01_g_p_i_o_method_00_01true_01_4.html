Detailed Description

template<class GPIOMethod>
class rur::AddressSwitch< GPIOMethod, true >

基板とかのアドレススイッチのクラス
さかさまにつけてしまった時用に逆にできるようにしたけど基板ごとに中身違うとあれなので多分使わない。

使用例
F3::AddressSwitch addressSwitch( { F3::PB6, F3::PB5, F3::PB4, F3::PB3 });
addressSwitch.config();
uint8_t address = addressSwitch.getAddress();
