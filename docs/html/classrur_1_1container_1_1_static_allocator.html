Detailed Description

template<class T, std::size_t capacity>
class rur::container::StaticAllocator< T, capacity >

スタック領域のみで動作するコンテナを作るためのアロケータ(非POD版)

決められたバッファサイズのスタック領域のみで動作するメモリアロケータです

     アロケータはスタック領域に連続した`0~maxSize-1`個分のメモリを確保します。@n
     またそれぞれのオブジェクト領域には'0~maxSize-1'のインデックスが付けられます。@n
     それぞれのメモリ領域に対して個別にオブジェクトの構築/破棄が行えます
計算量
操作 計算量
各要素への添え字アクセス O(1)
各要素の構築 O(1)
各要素の破棄 O(1)
Attention
他のコンテナからの使用を想定しているので、高速化のためにオブジェクトの重複構築/破棄忘れに関する管理を行っていません
また、このクラスのコピーコンストラクタ/ムーブコンストラクタでは何もしません(フラグ管理を行っていないため)上位のクラスで行ってください
Note
十分高速にメモリ使用のフラグ管理が行えるのならばこのクラス内で行うべきである
当クラスの作成時は単純な配列での実装と同等の処理速度が達成できることを目標としていたためです
Template Parameters
Tオブジェクトの型 非void && const修飾されてない && destructible を満たす必要があります
maxSizeオブジェクトの最大数 この数を超えてアロケートされることはありません
