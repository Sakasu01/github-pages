Detailed Description

template<class TIMMethod, class GPIOMethod>
class rur::VariableCycleTimer< TIMMethod, GPIOMethod, true, true >

可変周波数のタイマのクラス。

高周波のタイマ内で任意の周波数で関数を呼び出せるクラスです。 メンバ変数vector化したので一つのタイマでいくつも回せます。

使用例
F4::VariableCycleTimer tim = { F4::TimerClockParameter::generate<F4::tim8, 10000>(), callback };
tim.config();
tim.setStart(true);
tim.setFrequency(frequency);
peripheral::TIM::setStart
void setStart(bool start)
