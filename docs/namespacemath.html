値の2乗を計算する

この関数の需要低そうじゃないですか。でも実はそんなことないんですよ。

例えばこの場合(こんな計算が必要かはさておいて)

float x = sin(hogefunc() * hogefunc());
sin
EIGEN_DEVICE_FUNC const SinReturnType sin() const
Definition: ArrayCwiseUnaryOps.h:220

当然hogefuncが副作用を持つ場合や、hogefunc()の代わりにhogefunc() - 1等を二乗したい場合は次のようになる

float tmp = hogefunc() - 1;
float x = sin(tmp);

こういった場面ってかなり多いんですよ(経験則)。ここでこの関数の出番ですね

float x = sin(math::sq(hogefunc()));
