Detailed Description

template<class Derived>
class peripheral::I2CMethodInterface< Derived >

I2Cの操作インターフェースクラス

I2C<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

実装例
namespace F4 {
class I2CMethod : peripheral::I2CMethodInterface<I2CMethod> {
protected:
// 各メソッドの実装
};
}
