struct peripheral::TimerClockGenerator< T, periph, freq, period_ >

Deprecated:

Template Parameters
Tperipheral::Peripheral<>の特殊化をしたものとかを指定
periphperipheral::Peripheral<>のコンパイル時定数
frequencyタイマの周波数を設定します ex.)40kHzのPWMを生成したい->40'000
period_ピリオドに指定があるなら デフォルトで最大が取れるように計算されます
使用例
// tim1を100'000Hz periodは取りうる最大
TIM<F4::TIMMethod> tim1(F4::TimerClockGenerator<F4::tim1, 100000>{});
// tim2を50Hz periodを20'000
TIM<F4::TIMMethod> tim2(F4::TimerClockGenerator<F4::tim2, 50, 20000>{});
