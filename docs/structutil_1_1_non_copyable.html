Detailed Description

template<class T>
struct util::NonCopyable< T >

コピー禁止Mix-in

継承することで、継承先のクラスにコピー禁止属性を付与します
CRTPで継承してください

class Hoge : NonCopyable<Hoge> {}; // このクラスはコピー禁止になる
Hoge hoge1, hoge2;
// hoge1 = hoge2; // コンパイルエラー!!
Hoge hoge3(hoge1); // コンパイルエラー!!
