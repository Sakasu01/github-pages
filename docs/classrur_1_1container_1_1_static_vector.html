Detailed Description

template<class T, std::size_t capacity_>
class rur::container::StaticVector< T, capacity_ >

スタック領域のみで動作するstd::vectorに似たコンテナクラス

決められたバッファサイズ内のみで動作する動的配列みたいなやつです 各オブジェクトのメモリが'begin()'から始まり、連続することを保証します

Attention
std::vectorと互換を持たせるために、あえて部内のコーディング規約を無視しています
計算量
操作 計算量
各要素への添え字アクセス O(1)
全要素の両方向への操作 O(N)
末尾の要素の追加/削除 O(1)
コンテナのコピー/ムーブ O(N)
ぶっちゃけた話、動的アロケートのコストを気にしなければstd::vectorで十分です(moveとかのコストはあちらに軍配が上がる)
Note
std::vector::insert/eraseに相当するものがないので、そのうち作ったほうがいいと思う
Boost::static_vectorが使えればそれでいいのにな...
Template Parameters
Tオブジェクトの型 非void && const修飾されてない destructibleを満たす必要があります
capacity_オブジェクトの最大数 この数を超えて要素を確保することは出来ません
