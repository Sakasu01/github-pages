Detailed Description

自作コンテナ群

Static~コンテナ
基本的にc++ STLのコンテナと同じ動作をしますが、 すべてのコンテナで動的メモリ確保(=ヒープ領域の使用)を行いません(メモリはスタック領域に配置されます)
その代わり、コンテナサイズの自動拡張が起きないデータ構造となっています
See also
boost::containerのstatic_vector 等
StaticVector<uint32_t, 4> data;
data.pushBack(0);
data.pushBack(1);
data.pushBack(2);
data.pushBack(3);
data.pushBack(4); // メモリの際メモリの再確保などは起きずに無視される(エラー排出はする(詳細は各メソッドの説明へ))
