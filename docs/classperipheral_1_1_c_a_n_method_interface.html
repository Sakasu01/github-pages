template<class Derived>
class peripheral::CANMethodInterface< Derived >

CANの操作インターフェースクラス

CAN<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

namespace F4 {
class CANMethod : peripheral::CANMethodInterface<CANMethod> {
protected:
// 各メソッドの実装
};
}
