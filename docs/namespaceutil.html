正規化したインデックスを計算する

要素数Nの配列のインデックスは[0, N)の範囲だが、リングバッファなどで-1, N + 1 といったインデックスにアクセスしたい場合がある。
この関数はその問題を解決する。

Parameters

size配列サイズ 非ゼロであること
Returns
正規化インデックス [0, n)
Return values
