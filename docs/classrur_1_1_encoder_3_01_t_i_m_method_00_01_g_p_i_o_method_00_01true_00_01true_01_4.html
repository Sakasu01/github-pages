Detailed Description

template<class TIMMethod, class GPIOMethod>
class rur::Encoder< TIMMethod, GPIOMethod, true, true >

エンコーダのクラス

1回転あたりで正規化したやつとか計算できる
とりあえず4倍読み(各相のP/Nエッジで読み取る)

使用例
Encoder<F4::TIMMethod> encoder(F4::tim1, F4::PA0, F4::PA1);
encoder.config();
auto hoge = encoder.getIntegratedNormalizedValue();
rur::Encoder
TIM, GPIOの正しい継承になっていないときにを継承してないときにマッチするやつ 正しいクラスの時はSFINAEによって下のやつにマッチする
