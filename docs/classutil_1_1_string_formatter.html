フォーマット文字列を出力する

printfよりも安全に出力できます
printf系の'%~'の代わりに'$'が出力指定子です
'$'と引数の数が合わなくても、安全に出力されます

print("a = $", 12); // "a = 12"
print("a = $, b = $", 12, 3.4); // "a = 12, b = 3.4000"
print("a = $", 12, 3.4); // "a = 12" bは評価されない
print("a = $,b = $", 12); // "a = 12, b = $" 過剰な$はそのまま表示される
