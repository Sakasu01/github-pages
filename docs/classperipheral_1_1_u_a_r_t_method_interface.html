Detailed Description

template<class Derived, class DMAMethodChannelInfo>
class peripheral::UARTMethodInterface< Derived, DMAMethodChannelInfo >

UARTの操作インターフェースクラス

UART<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

Template Parameters
DMAMethodChannelTypeUARTで使用するDMAMethod::ChannelInfo型
Attention
Derived内でpublic指定でDMAMethodを定義すること UARTのSFINAEのために必要
実装例
namespace F4 {
class UARTMethod : peripheral::UARTMethodInterface<UARTMethod, DMAMethod::ChannelInfo> {
public:
using DMAMethod = F4::DMAMethod; // 必須
protected:
// 各メソッドの実装
};
}
