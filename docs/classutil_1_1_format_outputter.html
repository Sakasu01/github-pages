Detailed Description

template<class Derived>
class util::FormatOutputter< Derived >

フォーマット出力を行うCRTPクラス

出力方法として

  • '$'による安全なフォーマット
  • <<演算子によるストリーム出力もどき
  • c言語方式のフォーマット出力 を実装しています

但し、ストリーム出力は自前実装なので、std::endlやstd::hexといったマニピュレータは使えません

18/12/20 std::hex等書式指定の代替としてFOHelperを実装しました

出力データはDerived::appendTxBuffer()に転送されます

See also
print(std::string, Args...)
FOHelper
Template Parameters
DerivedCRTP継承クラス InputIterator appendTxBuffer(InputIterator first, InputIterator last)がアクセス可能であること
使用例
class SerialClass : public FormatOutputter<SerialClass> {
template <typename InputIterator>
InputIterator appendTxBuffer(InputIterator first, InputIterator last);
};
SerialClass serial;
serial << "a = " << a << "\n";
serial.print("b = $\n", b);
serial.printCStyleFormat("c = %d\n", c);

出力書式の指定
printfでいうところの%05dみたいな書式を指定するには、FOHelperを用います
FOHelperはストリーム出力とprintメソッドによる出力にのみ対応しています
整数と浮動小数は引数から自動判別されます
serial << FOHelper<5, '0'>(12) << "\n"; // "%05d"と同じフォーマット
serial.print("$\n", FOHelper<9, 5>(12.5f)); // "%9.5f"と同じフォーマット
まだ調整途中の機能なので、使いずらくても悪しからず
See also
FOHelper
出力形式の追加
以下のような記述を行ことで、任意の形式で出力ができます
struct HogeClass {
int huga;
};
template <class T>
util::FormatOutputter<T>& operator<<(util::FormatOutputter<T> &stream, HogeClass x) {
return stream << x.huga;
}

フォーマット文字列を出力する

printfよりも安全に出力できます
printf系の'%~'の代わりに'$'が出力指定子です
'$'と引数の数が合わなくても、安全に出力されます

print("a = $", 12); // "a = 12"
print("a = $, b = $", 12, 3.4); // "a = 12, b = 3.4000"
print("a = $", 12, 3.4); // "a = 12" bは評価されない
print("a = $,b = $", 12); // "a = 12, b = $" 過剰な$はそのまま表示される

Attention
現在、'%'に相当する'$'のエスケープ方法はありません
