Detailed Description

template<class TIMMethod, class GPIOMethod>
class rur::Stepper< TIMMethod, GPIOMethod, true, true >

ステッパーのクラス。

現段階で可変な周期のタイマを生成できないため、 高周期のタイマ割り込みでupdateを呼び出すことにより疑似的に可変な周期のタイマを生成します。 なるべく高い周期のタイマにしてください。

使用例
constexpr uint32_t stepperTimerFreq = 100000;
F3::Stepper stepper = {stepperTimerFreq, F3::PA12, F3::PB4, F3::PA9};
F3::TIM stepperTim = F3::TimerClockParameter::generate<F3::tim1, stepperTimerFreq>();
void updateStepper() {
stepper.update();
}
stepper.config();
stepperTim.config();
stepperTim.configIT(F3::TIM::InterruptMode::update, 0, updateStepper);
stepperTim.setStart(true);
