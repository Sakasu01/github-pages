Detailed Description

template<class Derived>
class peripheral::ADCMethodInterface< Derived >

ADCの操作インターフェースクラス

ADC<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

Attention
Derived内でpublic指定でGPIOMethodを定義すること
実装例
namespace F4 {
class ADCMethod : peripheral::ADCMethodInterface<ADCMethod> {
public:
using GPIOMethod = F4::GPIOMethod; // 必須
protected:
// 各メソッドの実装
};
}
