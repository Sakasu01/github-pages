Detailed Description

template<class Derived>
class peripheral::TIMMethodInterface< Derived >

TIMの操作インターフェースクラス

TIM<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はのメソッドアクセス指定をprotected以上にすること(protected推奨)

Attention
Derived内でpublic指定でChannelMethodを定義すること
実装例
namespace F4 {
class TIMMethod : peripheral::TIMMethodInterface<TIMMethod> {
public:
class ChannelMethod; // 必須
protected:
// 各メソッドの実装
};
class TIMMethod::ChannelMethod : peripheral::TIMChannelMethodInterface<ChannelMethod> {
protectd:
};
}
