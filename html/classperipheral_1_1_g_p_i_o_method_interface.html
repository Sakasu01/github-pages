Detailed Description

template<class Derived>
class peripheral::GPIOMethodInterface< Derived >

GPIOの操作インターフェースクラス

GPIO<Method>のMethodとして使用されるクラスのインターフェース
CRTP継承によってマイコンごとの操作を実装すること

実装の際はメソッドアクセス指定をprotected以上にすること(protected推奨)

実装例
namespace F4 {
class GPIOMethod : peripheral::GPIOMethodInterface<GPIOMethod> {
protected:
// 各メソッドの実装
};
}
